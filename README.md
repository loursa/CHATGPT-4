## 什么是镜像网站？
镜像网站（Mirror Site）是指一个网站的内容被复制并托管在另一个服务器上，形成与原网站内容几乎完全相同的网站。这些镜像网站通常使用不同的域名或服务器地址，但与原网站提供相同的内容或功能。
